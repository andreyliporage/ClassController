# Class Manager

<p>Class Manager é um desafio do Bootcamp Launchbase onde é possível adicionar professores e alunos, podendo
criar, atualizar e deletar os membros.</p>

### Ferramentas utilizadas:

<ul>
    <li>HTML</li>
    <li>CSS</li>
    <li>Javascript</li>
    <li>NodeJs</li>
    <li>Express</li>
    <li>File System</li>
</ul>
